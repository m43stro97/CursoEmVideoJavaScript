Aula 5 - Variáveis e Tipos Primitivos

No JavaScript moderno, para variáveis podemos usar o var ou let.

Identificadores:

  • Podemo começar com letra, $ ou _
  • Não podem começar com números
  • É possivel usar letras ou números
  • É possivel usar acentos e símbolos
  • Nao podem conter espaços
  • Nao podem ser palavras reservadas. Ex: var, null

Dicas

  • Maiúsculas e minúsculas fazem diferença
  • Tentar escolher nomes coerentes para as variáveis

Tipos de Dados Primitivos

  • Number
    • Infinito
    • NaN
  • String
  • Boolean
  • Null
  • Undefined
  • Object
    • Array
  • Function

Tipos de conversão:

Para converter uma string em number podemos usar o Number.parseInt() para converter para inteiro, e podemos usar o Number.parseFloat() para converter para float. Atualmente podemos usar a conversão de duas maneiras: só Number, ele assume para qual tipo tem de converter, ou so parseInt() e parseFloat que ele assume que é do tipo Number.

Formatação de strings:

var s = 'JavaScript'

'Eu estou a aprender s' //nao faz interpolação

'Eu estou a aprender ' + s //faz concatenação

`Eu estou a aprender ${s}` //usa template String

s.length //quantos caracteres tem a string

s.toUpperCase() //tudo para 'Maiusculas'

s.toLowerCase() //tudo para 'Minusculas'

Formatação de numeros:

var n1 = 1540.5

n1.toFixed(2) //mete o n1 com duas casa decimais

n1 //1540.50

n1.toFixed(2).replace('.',',') // mete o n1 = 1540,50
