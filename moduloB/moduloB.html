Aula 5 - Variáveis e Tipos Primitivos

No JavaScript moderno, para variáveis podemos usar o var ou let.

Identificadores:

  • Podemo começar com letra, $ ou _
  • Não podem começar com números
  • É possivel usar letras ou números
  • É possivel usar acentos e símbolos
  • Nao podem conter espaços
  • Nao podem ser palavras reservadas. Ex: var, null

Dicas

  • Maiúsculas e minúsculas fazem diferença
  • Tentar escolher nomes coerentes para as variáveis

Tipos de Dados Primitivos

  • Number
    • Infinito
    • NaN
  • String
  • Boolean
  • Null
  • Undefined
  • Object
    • Array
  • Function
